# Gerador de Pedidos

Aplicação desenvolvida em Python para gerar pedidos de forma automatizada, com foco em reduzir erros manuais e agilizar o processo de criação de pedidos.

O sistema foi pensado para uso prático em um cenário real, priorizando simplicidade e eficiência.

## Funcionalidades
- Geração automática de pedidos
- Padronização das informações geradas
- Execução via interface gráfica simples (arquivo `.pyw`)

## Tecnologias
- Python
- Git

## Como executar
1. Certifique-se de ter o Python instalado
2. Execute o arquivo:
```bash
python GeradorDePedidos.pyw
